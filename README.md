
# 🌍 GeoVault — Explore, compare, and understand countries worldwide 

GeoVault is a modern, responsive web application that allows users to explore detailed information about countries worldwide, including population, capital, region, languages, and flags.The app includes interactive features such as search, filtering, sorting (using array higher-order functions), country comparison, and a dark/light mode toggle to enhance user experience.

---

## 📌 Project Overview


Built using **HTML, Vanilla JavaScript, and Tailwind CSS**, the application fetches real-time data from the **REST Countries API** to display key details such as population, capital, region, languages, and flags.

GeoVault focuses on providing a smooth user experience through dynamic features like **search, filtering, and sorting**, all implemented using JavaScript array higher-order functions (`filter`, `sort`, `map`, etc.). 

The goal of this project is to demonstrate strong fundamentals of **API integration, DOM manipulation, and modern UI design**, while building a practical and visually appealing data-driven application.

## ✨ Features


* **Country Info Card** — Displays population, region, capital, languages, currency

* **Search Functionality** — Implemented using `Array.filter()`
* **Filtering System** — Region, subregion, population filters using chained `filter()`
* **Sorting Options** — Name, population, area using `Array.sort()`
* **Comparison Panel** — Compare up to 2 countries side-by-side
* **Dark/Light Mode** — Toggle for better user experience
